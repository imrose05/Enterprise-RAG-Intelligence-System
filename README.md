## 📖 Overview

Enterprise RAG Intelligence System is a full-stack AI application that demonstrates how Retrieval-Augmented Generation (RAG) can be combined with Role-Based Access Control (RBAC) to build secure enterprise AI assistants. Built using Node.js, Express.js, Anthropic Claude Sonnet 4.6, HTML, CSS, and Vanilla JavaScript, the system enables users to query enterprise data using natural language while ensuring access is restricted based on their roles.

## ✨ Features

* 💬 Natural language enterprise query interface
* 🔍 Retrieval-Augmented Generation workflow simulation
* 🔐 Role-Based Access Control with 5 enterprise roles
* ⚡ Real-time RAG pipeline visualization
* 🧭 Intelligent source routing
* 🚫 Access-denied handling without data leakage
* 🔄 Live role switching for demonstrations
* 🛡️ Secure server-side API architecture

🚀 Installation
git clone https://github.com/imrose05/Enterprise-RAG-Intelligence-System.git
cd Enterprise-RAG-Intelligence-System
npm install

Create a .env file:
ANTHROPIC_API_KEY=your_api_key
PORT=3000
To Start the server:node server.js
